# Action mapping (Reacher-v5)
# Action space: Box(-1, 1, (2,), float32) — torques at hinges.
#
# Index → Joint (env action order, reacher.xml):
#   [0] joint0 — torque at first hinge (fixture → first link)
#   [1] joint1 — torque at second hinge (first link → second link / fingertip)
#
# Notes:
# - Reward encourages small fingertip−target distance: negative norm of COM offset plus quadratic action penalty.
# - Target position (qpos[2:4]) is resampled on reset; not directly actuated by the 2-dim action.

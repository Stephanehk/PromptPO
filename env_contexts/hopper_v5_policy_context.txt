# Action mapping (Hopper-v5)
# Action space: Box(-1, 1, (3,), float32) — torques applied at hinge joints (scaled by actuators in MuJoCo).
#
# Index → Joint name in hopper.xml (env action order):
#   [0] thigh_joint — torque between torso and thigh (N·m nominal scale from [-1,1])
#   [1] leg_joint   — torque between thigh and leg
#   [2] foot_joint  — torque between leg and foot
#
# Notes:
# - Each component ∈ [-1, 1] is a normalized torque command; the simulator maps to actuator effort.
# - The root slides along x (forward) and z (height); pitch of torso is rooty; three leg hinges produce hopping.
# - Forward reward uses horizontal displacement of qpos[0] (rootx) per step; see Gymnasium hopper_v5.py.

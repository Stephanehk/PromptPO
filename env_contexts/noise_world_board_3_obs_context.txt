# Observation — env_name noise_world_board_3

At each step you receive a `NoiseWorldObservation` instance
(class in direct_policy_learning.observations.noise_world_observation).

The raw observation vector has length 4 + n * n where n is the grid side length
(for the fixed evaluation boards under this env_name, n = 10).

**Positions** — scalar floats in [0, 1], normalized by the environment (row/col divided by max(n-1, 1)):

- `agent_row`, `agent_col` — current agent cell
- `goal_row`, `goal_col` — goal cell (fixed for the environment instance)

**Layout** — attribute `board`: a flat sequence of length n * n in row-major order (cell (r, c) is index r * n + c). The same data can be arranged as a 2D array of shape (n, n) with grid row r as row r of that matrix. Entries are integer labels fixed for the environment instance.

**Board cell values** — only the integers 0, 1, 2, 3, 4, and 5 can appear as entries of `board`; no other values occur.

# Action mapping (HalfCheetah-v5)
# Action space: Box(-1, 1, (6,), float32) — torques applied at hinge joints.
#
# Index → Joint (env action order):
#   [0] bthigh    — back thigh joint (torso ↔ back thigh)
#   [1] bshin     — back shin joint (back thigh ↔ back shin)
#   [2] bfoot     — back foot joint (back shin ↔ back foot)
#   [3] fthigh    — front thigh joint (torso ↔ front thigh)
#   [4] fshin     — front shin joint (front thigh ↔ front shin)
#   [5] ffoot     — front foot joint (front shin ↔ front foot)
#
# Notes:
# - Each value ∈ [-1, 1] is a torque command (scaled internally by actuator strength).
# - "b" = back leg, "f" = front leg.
# - Each leg has 3 joints: thigh (hip), shin (knee), foot (ankle).
# - Locomotion emerges from coordinated extension/flexion of back and front legs.
